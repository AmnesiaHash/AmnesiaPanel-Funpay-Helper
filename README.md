# AmnesiaPanel — FunPay Helper

**Desktop-хелпер для FunPay**, заточенный под **массовый залив и управление лотами** за считанные секунды.

Бесплатное open-source приложение для Windows. Без подписок и без внешних серверов лицензий — всё работает локально на вашем ПК.

[Скачать установщик (Releases)](https://github.com/AmnesiaHash/AmnesiaPanel-Funpay-Helper/releases/latest) · [Telegram-канал](https://t.me/AmnesiaPanel) · [Автор](https://t.me/AmnesiaOwner)

---

## Зачем это нужно

Если вы ведёте много лотов на FunPay, руками править цены, описания и статусы — долго.  
**AmnesiaPanel** подключает ваш аккаунт по Golden Key и даёт панель управления: массовые правки, шаблоны, копирование лотов и быстрый повторный залив.

---

## Скачать и установить

1. Откройте **[Releases](https://github.com/AmnesiaHash/AmnesiaPanel-Funpay-Helper/releases/latest)**
2. Скачайте `AmnesiaPanel-Setup-….exe`
3. Установите и запустите
4. Введите FunPay **Golden Key** на экране подключения

> Обычным пользователям исходники не нужны — достаточно установщика из Releases.

## Возможности

### Подключение
- Вход по FunPay Golden Key
- Ключ хранится **только на вашем ПК** (шифрование Electron `safeStorage` + локальная SQLite)
- Уходит **только** на FunPay как cookie — не на сторонние сервера

### Главная
- Обзор аккаунта, баланса, числа лотов и категорий
- Быстрый старт по разделам приложения

### Лоты
- Загрузка списка лотов с FunPay
- Поиск, фильтры и сортировка
- Массовое изменение **цены**, **названия**, **описания**, **статуса** (вкл/выкл)
- Сохранение лота (или нескольких) в **шаблон**
- Копирование лотов в другие категории / игры

### Шаблоны
- Библиотека сохранённых лотов для повторного залива
- Массовое выставление шаблонов на FunPay
- Импорт / экспорт шаблонов
- Удаление по одному или пачкой

### История
- Журнал операций: загрузка лотов, массовые обновления, шаблоны, импорт/экспорт

### Настройки
- Темы: тёмная / светлая / системная
- Смена Golden Key и проверка соединения
- Очистка локальных данных

## Сборка из исходников

Нужны **Node.js 20+** и **Windows x64**.

```bash
npm run setup
```

Поставит зависимости, соберёт `AmnesiaPanel.exe` и создаст ярлык на рабочем столе.

```bash
npm run dev      # разработка
npm run dist     # установщик NSIS → папка release/
```
